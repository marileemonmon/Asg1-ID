# Asg1-ID

Github Link: https://github.com/marileemonmon/Asg1-ID
Github Pages: https://marileemonmon.github.io/Asg1-ID/

TXT Concert Booking Website
    -A single-page TXT concert booking website using HTML, CSS, JS. Includes artist details, ticket info, venues, sign-in forms, and a payment simulation.

User Audience
    -TXT fans & users searching for concert info.

Purpose
    -To provide artist information, ticket prices, venue details, and a simple booking process.

Features
    -Home images

    -Artist bio + photos + video + Instagram link

    -Ticket prices

    -Card details form + Apple Pay alert

    -Sign In / Sign Up

    -Venue list
